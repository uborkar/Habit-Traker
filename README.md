# 🎯 Habit Tracker - Your Path to Success

A simple yet powerful habit tracking web application to help you track your 24-hour activities and visualize your journey from zero to success.

## 🚀 Vision

This project is designed to be a comprehensive personal development tool that helps ambitious individuals track their habits, analyze their time usage, and optimize their daily routines. The ultimate goal is to create a complete ecosystem including:

- ✅ **Web Application** (Current Phase)
- 📱 **Android App** (Future)
- 🤖 **Personal Intelligent Trainer/Guide** (Future)
- 📊 **Advanced Analytics & Insights** (Future)

## 🌟 Features

### Current Features (v1.0)

- **24-Hour Activity Tracking**: Log all your daily activities with start and end times
- **Category Organization**: Organize activities by category (Productive, Health, Learning, Work, Social, Rest, Other)
- **Visual Analytics**: 
  - Doughnut chart showing time distribution by category
  - Bar chart displaying productive vs non-productive hours
- **Real-time Statistics**:
  - Total activities logged
  - Total time tracked
  - Productive time
  - Success score (percentage of productive time)
- **Activity Timeline**: View all your activities in chronological order
- **Data Persistence**: All data is saved locally in your browser
- **Data Export**: Export your daily data as JSON for backup or analysis
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices

## 📋 How to Use

### Getting Started

1. **Open the Application**: Simply open `index.html` in your web browser
2. **Track an Activity**:
   - Enter the activity name
   - Set start and end times
   - Select a category
   - Add optional notes
   - Click "Add Activity"
3. **View Your Progress**: Check your daily summary and charts
4. **Export Data**: Click "Export Data" to download your activities as JSON

### Activity Categories

- **Productive**: General productive activities
- **Health & Fitness**: Exercise, yoga, sports, etc.
- **Learning**: Reading, courses, skill development
- **Work**: Professional work and tasks
- **Social**: Social interactions and networking
- **Rest & Sleep**: Sleep and rest periods
- **Other**: Miscellaneous activities

### Success Metrics

The app calculates a **Success Score** based on:
- Time spent on productive activities (Productive, Learning, Work, Health)
- Percentage of the 24-hour day used productively
- Higher scores indicate better time utilization

## 📊 Data Management

### Local Storage
- Activities are automatically saved to browser's LocalStorage
- Data persists between sessions
- Each activity includes: name, time, duration, category, notes, and date

### Data Export
- Export daily data as JSON
- Use exported data for backup or further analysis
- File naming: `habit-tracker-YYYY-MM-DD.json`

### Clear Data
- Clear all activities for the current day
- Confirmation dialog prevents accidental deletion
- Previous days' data remains intact
